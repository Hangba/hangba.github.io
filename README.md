## 🚀 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/stevehangba/hangba.github.io.git
   cd hangba.github.io
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

4. Build for production:
   ```bash
   pnpm build
   ```
